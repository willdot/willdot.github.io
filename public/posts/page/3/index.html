Get Docker container from Azure and run locally

So I had a Docker image on an Azure container registry, and I wanted to pull it and run it locally on my machine. So with command line I did the following: Start Docker on my machine. (I always forget this….) Login to the Azure registry. sudo docker login <servername> -u <username> -p <password> Pull the image that I want. (Tag is optional to get a specific version). docker pull <servername>/<registry>/<image>:<tag> Take a look at the images locally so that I can get the image id to start a container.
